# koajs-challenge

Simple CRUD RESTAPI based on koajs

Database: SQLite<br />
Caching: node-cache

## Installation

cd project folder and install node modules

```bash
npm install
```
You can remove sqlite.db file from the project folder and create new one

```bash
npm run migrate
```

## Usage

```bash
node index.js
```

You can use Postman to test API. See postman folder.
